🔍 ROOT CAUSE ANALYSIS
═══════════════════════════════════════════════════════════════════════════════

ROOT CAUSE 1: Alpine.js Scope Missing
  - welcome.blade.php menggunakan Alpine.js dengan dashboard() function
  - dashboard() mendefinisikan: showToast, toastType, toastMessage
  - layouts/app.blade.php (untuk admin pages) TIDAK punya Alpine.js data
  - Template di admin pages mencoba akses variabel yang tidak ada
  - Browser console spam error setiap kali render element dengan x-show/x-text

ROOT CAUSE 2: CSS Specificity
  - .menu-item:hover dan .menu-item.active digabung dalam satu rule
  - Hover override active state
  - Background gradient terlalu terang (tidak kontras dengan sidebar hijau)

SOLUSI 1: Global Alpine.js Data untuk Admin Pages

File: resources/views/layouts/app.blade.php

A. Tambah Alpine.js CDN:
   <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

B. Definisikan adminApp() component:
   ```javascript
   document.addEventListener('alpine:init', () => {
       Alpine.data('adminApp', () => ({
           showToast: false,
           toastMessage: '',
           toastType: 'info',
           
           displayToast(message, type = 'info') {
               this.toastMessage = message;
               this.toastType = type;
               this.showToast = true;
               setTimeout(() => { this.showToast = false; }, 5000);
           }
       }));
   });
   ```

C. Attach ke body tag:
   <body x-data="adminApp()">

BENEFIT:
  ✓ Semua halaman admin punya akses ke showToast/toastType/toastMessage
  ✓ No more console errors
  ✓ Toast notification siap pakai untuk AJAX feedback
  ✓ Backward compatible dengan welcome.blade.php (dashboard)

SOLUSI 2: Sidebar Active State Enhancement

File: resources/views/layouts/app.blade.php (CSS section)

SEBELUM (Masalah):
```css
.menu-item:hover,
.menu-item.active {
    background: linear-gradient(90deg, #bbf7d0 0%, #22c55e 100%);
    color: #166534;
    font-weight: 700;
    box-shadow: 0 2px 8px rgba(34,197,94,0.08);
}
.menu-item:hover {
    background-color: rgba(255,255,255,0.13);
    color: #fff;
}
```
Issues: Hover override active, warna active kurang kontras

SESUDAH (Fixed):
```css
.menu-item.active {
    background: linear-gradient(90deg, #ffffff 0%, #dcfce7 100%);
    color: #166534;
    font-weight: 700;
    box-shadow: 0 4px 12px rgba(34,197,94,0.2);
    transform: translateX(2px);
}

.menu-item:hover {
    background-color: rgba(255,255,255,0.13);
    color: #fff;
}
```

IMPROVEMENTS:
  ✓ Active state terpisah dari hover (specificity clear)
  ✓ Background putih ke hijau muda (#ffffff → #dcfce7)
  ✓ Shadow lebih dalam (0.2 opacity vs 0.08)
  ✓ Transform translateX(2px) untuk depth effect
  ✓ Hover tetap semi-transparent (tidak ganggu active)
