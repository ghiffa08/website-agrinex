🚀 NEW ENDPOINTS:

   GET /api/v1/dashboard/poll?last_update={timestamp}
   └─ Full data dengan conditional request (hemat bandwidth 80%)

   GET /api/v1/dashboard/poll-status?last_update={timestamp}
   └─ Lightweight status only (< 1KB)

✨ KEY FEATURES:

   Backend:
   ✓ Conditional request (hanya kirim data jika berubah)
   ✓ Cache-based change detection
   ✓ Rate limiting (4 req/min)
   ✓ No persistent connections needed

   Frontend:
   ✓ Auto-start dengan data-dashboard-polling attribute
   ✓ Auto-pause saat tab hidden (hemat bandwidth)
   ✓ Custom event 'dashboard:updated' untuk Alpine.js
   ✓ Configurable interval (default 20 detik)
   ✓ Error handling & retry logic
