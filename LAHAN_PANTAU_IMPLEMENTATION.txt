API ENDPOINTS:
- GET    /api/v1/lahan-pantau           → List semua lahan dengan total devices
- POST   /api/v1/lahan-pantau           → Create lahan baru
- GET    /api/v1/lahan-pantau/{id}      → Detail lahan dengan list devices
- PUT    /api/v1/lahan-pantau/{id}      → Update lahan
- DELETE /api/v1/lahan-pantau/{id}      → Delete lahan (unassign devices dulu)

WEB ROUTES:
- GET /lahan-pantau       → Halaman daftar lahan (index)
- GET /lahan-pantau/{id}  → Halaman detail lahan (belum dibuat)
